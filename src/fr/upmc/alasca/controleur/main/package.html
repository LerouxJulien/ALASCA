Contient les classes permettant de lancer la simulation du centre de calcul.

Chaque classe a des paramètres de simulation différents. Le nombre de pc et le nombre de coeurs par pc sont les paramètres sur lesquels on joue.

Dans les 3 simulations, 2 applications sont déployées sur le centre de calcul ; le générateur de requêtes envoie des requêtes pour ces 2 applications plus une autre non prise en charge.
