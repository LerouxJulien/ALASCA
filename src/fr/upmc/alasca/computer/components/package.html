Contient les classes Computer et VirtualMachine.

Le Computer est chargé de déployer une nouvelle machine virtuelle si le contrôleur en fait la demande. Il sera aussi plus tard chargé de détruire les machines virtuelles.

Les machines virtuelles traitent les requêtes reçues. Chaque machine virtuelle a au minimum 2 coeurs. Les coeurs sont représentés par la classe interne VMThread.

Chaque requête met un certain temps à être traiter : le calcul du temps de traitement d'une requête est simplement la division du nombre d'instructions d'une requête par la fréquence du coeur qui la traite.
